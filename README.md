# LDH-evo
Evolutionary dynamics of LDH genes in mammals.

The instructions for the code is contained within the R/ python files and includes all analyses (apart from those performed on Galaxy Australia). The OrthoFinder results are excluded from the available data given that the data can easily be generated by downloading the listed genomes and running OrthoFinder. Otherwise data can be sourced within this GitHub or by following the instructions within the main text. 

The comparative genomics analyses were split across the ortho-process.py file (phylogeny/ ortholog processing), non-phylo.py file (CDS retrieval), LDH.py (dN/dS production) and the evolution.Rmd file (visualisation and dN/dS tree dating).  

The American Pika code was split between the climate_change.py file (searching for lactate dehydrogenase SNPs) and the aging.py file ('aging' the lineages by the presence/absence of SNPs). 

The Bank Vole code was located within the Bank-Vole.py file.
