# LDH-evo
Evolutionary dynamics of LDH genes in mammals.

Results of OrthoFinder are within the OrthoFinder folder. The instructions for the code is contained within the R/ python files and includes all analyses (apart from those performed on Galaxy Australia). 

The comparative genomics analyses were split across the ortho-process.py file (phylogeny/ ortholog processing), non-phylo.py file (dN/dS and CDS retrieval) and the evolution.Rmd file (visualisation and dN/dS tree dating).
